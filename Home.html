Tecnologias

CSS

¿Que es CSS?

Las siglas CSS (Cascading Style Sheets) significan «Hojas de estilo en cascada» y parten de un concepto simple pero muy potente: aplicar estilos (colores, formas, márgenes, etc...) a uno o varios documentos (generalmente documentos HTML, páginas webs) de forma automática y masiva.

SASS

¿Qué es SASS

Syntactically Awesome Stylesheets en resumen «SASS es un preprocesador CSS. Funciona como una extensión de CSS que le permite usar cosas como variables, reglas anidadas, mixins, inline import, herencias y más, lo que hace que el código CSS sea mucho más eficiente.

GIT

¿Qué es GIT?

Git es una herramienta que realizar el sistema de control de versiones de código de forma distribuida. Es de código abierto, con mantenimiento activo y la herramienta de este tipo más empleada en el mundo.
